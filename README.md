A comprehensive learning repository for mastering the MERN (MongoDB, Express.js, React, Node.js) stack. This project serves as a hands-on guide to building full-stack JavaScript applications.

## 📚 What is MERN Stack?

The MERN stack is a popular JavaScript stack for building modern web applications:
- **M**ongoDB - NoSQL database
- **E**xpress.js - Backend web framework
- **R**eact - Frontend library
- **N**ode.js - JavaScript runtime environment

## 🚀 Technologies Covered

- **Node.js** - JavaScript runtime for server-side programming
- **Express.js** - Web application framework for Node.js
- **MongoDB** - NoSQL database with Mongoose ODM
- **RESTful APIs** - Building scalable API endpoints
- **Authentication** - JWT, bcrypt for password hashing
- **CRUD Operations** - Create, Read, Update, Delete operations
- **Error Handling** - Best practices for error management
- **Environment Variables** - Configuration management with dotenv
